BUMN Milik Indonesia

BUMN Milik Belanda

Kereta api pertama di Indonesia dibangun pada tahun 1867, menghubungkan Semarang dengan Tanggung oleh Nederlandsch-Indische Spoorweg Maatschappij (NIS). Perkembangan jalur kereta api terus berlanjut dengan pembangunan jalur-jalur baru yang menghubungkan berbagai kota di Jawa dan Sumatera. Pada masa kemerdekaan, kereta api menjadi sarana transportasi utama masyarakat. Era 1980-an ditandai dengan elektrifikasi jalur kereta api dan penggunaan kereta api listrik. Saat ini, PT Kereta Api Indonesia (KAI) mengoperasikan layanan kereta api penumpang dan barang di seluruh Indonesia.
